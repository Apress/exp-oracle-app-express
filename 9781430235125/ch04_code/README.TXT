Short explanation of the code snippets provided (used in the chapter on Team Development)

Add-Image-to-Apex-Builder.txt        : 
Contains a piece of Javascript to "tweak" the APEX Builder by adding an addtional image with a link.

APEX_TEAM_FEATURES_query.sql         : 
Query to create a Gantt Chart for the Features entered.

Custom Chart XML using P3_XML_PG.xml : 
Customized XML for the Gantt Chart.

GenerateProjectXML.sql               :
Function used to generate the XML for the Gantt chart

APEX_TEAM_FEEDBACK_query.sql         :
Query to show the entered Feedback.

send_feedback-Procedure.sql          :
Procedure to send feedback by mail.
